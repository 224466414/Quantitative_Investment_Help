ICIRclass.save_result()

ICIRclass.save_result(path)

  1. 释义

    存储计算出的IC与IR

  2. 参数

    path

    • 释义
      计算结果存储的文件夹位置
    • 类型
      字符型(str)

  3. 返回

    无

  4. 例子

    • 输入

      import pandas as pd
import sys
sys.path.append('modules')
from factorTest import ICIRclass

factor = pd.read_csv('factor/IVCAPM.csv').set_index('secIDs')
EReturn = pd.read_csv('factor/EReturn.csv').set_index('secIDs')

beginDate = '20050101'
endDate = '20210531'
confidence = 0.95

ICIRClass = ICIRclass(factor, EReturn, beginDate, endDate, confidence)
ICIRClass.get_result()

path = 'result/factor'
ICIRClass.save_result(path)

    • 输出

      在 'result/factor/IC' 文件夹中能够找到:
'ICMonth.csv': IC月度数据
'ICSummary.csv': IC总结数据
'ICFigure.png': IC时序图
