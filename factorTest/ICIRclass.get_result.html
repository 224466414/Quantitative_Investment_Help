ICIRclass.get_result()

ICIRclass.get_result()

  1. 释义

    根据创建ICIR法检验对象时输入的因子值数据、期望收益率数据计算各月的IC与IR以及一系列检验结果。

  2. 参数

    无

  3. 返回

    该方法将计算结果进行在ICIRclass的属性中存储而非返回，可通过访问属性进行提取。
    可提取的结果见5.结果

  4. 例子

    • 输入

      import numpy as np
import sys
sys.path.append('modules')
from factorTest import ICIRclass

factor = np.load('factor/方正证券_003/SmartMoney.npy')

beginDate = 20050101
endDate = 20210531
confidence = 0.95
frequency = 1

ICIRClass = ICIRclass(factor, beginDate, endDate, confidence, frequency)
ICIRClass.get_result()

print(ICIRClass.IC)
print(ICIRClass.summary)
