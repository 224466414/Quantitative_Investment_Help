ICIRclass.get_result()

ICIRclass.get_result()

  1. 释义

    根据创建ICIR法检验对象时输入的因子值数据、期望收益率数据计算各月的IC与IR以及一系列检验结果。

  2. 参数

    无

  3. 返回

    该方法将计算结果进行在ICIRclass的属性中存储而非返回，可通过访问属性进行提取。
    可提取的结果见5.结果

  4. 例子

    • 输入

      import pandas as pd
import sys
sys.path.append('modules')
from factorTest import ICIRclass

factor = pd.read_csv('factor/IVCAPM.csv').set_index('secIDs')
EReturn = pd.read_csv('factor/EReturn.csv').set_index('secIDs')

beginDate = '20050101'
endDate = '20210531'
confidence = 0.95

ICIRClass = ICIRclass(factor, EReturn, beginDate, endDate, confidence)
ICIRClass.get_result()

print(ICIRClass.IC)
print(ICIRClass.summary)
