groupclass.get_result()

groupClass.get_result()

  1. 释义

    根据创建分层法检验对象时输入的因子值数据、期望收益率数据计算各月的各组投资组合期望收益率。

  2. 参数

    无

  3. 返回

    该方法将计算结果进行在groupclass的属性中存储而非返回，可通过访问属性进行提取。
    可提取的结果见5.结果

  4. 例子

    • 输入

      import numpy as np
import sys
sys.path.append('modules')
from factorTest import groupclass

factor = np.load('factor/方正证券_003/SmartMoney.npy')

beginDate = 20050101
endDate = 20210531
groupNumber = 10
benchmark = '000905.SH'
frequency = 1

groupClass = groupclass(factor, beginDate, endDate, groupNumber, benchmark, frequency, ascending=True)
groupClass.get_result()

print(groupClass.groupNevDaily)
print(groupClass.summary)
