量化投资程序 帮助文档

regressionclass.save_result()

regressionClass.save_result(pathResult, regressionResult)

  1. 释义

    存储计算出的IC与IR。

  2. 参数

    pathResult

    • 释义
      计算结果存储的文件夹位置
    • 类型
      字符型(str)

    regressionResult

    • 释义
      回归法的计算结果
    • 类型
      数据框(pd.DataFrame)

  3. 返回

    无

  4. 例子

    • 输入

      import pandas as pd
import sys
sys.path.append('modules')
from factorTest import regressionclass

factor = pd.read_csv('factor/IVCAPM.csv').set_index('secIDs')
EReturn = pd.read_csv('factor/EReturn.csv').set_index('secIDs')

beginDate = '20050101'
endDate = '20210531'

regressionClass = regressionclass(factor, EReturn, beginDate, endDate)
regressionResult = regressionClass.get_result()

pathResult = 'result/factor'
regressionClass.save_result(pathResult, regressionResult)

    • 输出

      在 'result/factor/regression' 文件夹中能够找到:
'regression.csv': 回归数据
