groupclass()

from factorTest import groupclass  # 从factorTest文件中导入groupclass对象
groupClass = groupclass(beginDate, endDate, factor, groupNumber, ascending=True, frequency=1)  # 创建groupclass对象

  1. 释义

    创建分层法检验对象。必须要先创建该对象，才可通过其附带的各类方法进行分层法检验。

  2. 参数

beginDate

- 释义  
    检验开始的日期
- 类型  
    字符型(str)

endDate

- 释义  
    检验结束的日期
- 类型  
    字符型(str)

factor

- 释义  
    因子值数据
- 类型  
    数据框(pd.DataFrame)

groupNumebr

- 释义  
    分组的组数
- 类型  
    整型(int)

ascending (optional)

- 释义  
    检验因子的排序顺序，默认为True。若因子大小与期望收益负相关，则设定为True；若因子大小与期望收益正相关，则设定为False
- 类型  
    布尔型(bool)

frequency (optional)

- 释义  
    进行调仓的周期天数。如果设定为1则进行日频检验
- 类型  
    整型(int)

  1. 返回

    分层法检验对象

  2. 例子

    • 输入

      import pandas as pd
import sys
sys.path.append('modules')
from factorTest import groupclass

factor = pd.read_csv('factor/IVCAPM.csv').set_index('secIDs')

beginDate = '20050101'
endDate = '20210531'
groupNumber = 5

groupClass = groupclass(beginDate, endDate, factor, groupNumber, ascending=True, frequency=1)

      无
