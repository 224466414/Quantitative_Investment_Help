neutralize()

neutralize(data)

  1. 释义

    使用Box-Cox变换将因子转换为近似正态分布

  2. 参数

    data

    • 释义
      输入的因子数据。需要注意，输入的因子数据必须为正数。通常可使用单调变化，如因子值+某一常数，将因子数据转为正数。
      需要以数据框(pd.DataFrame)格式输入，并且保证其index为证券代码，columns为交易月份/日期。
    • 类型
      数据框(pd.DataFrame)

  3. 返回

    • 释义
      因子值经过正态化后的数据
    • 类型
      数据框(pd.DataFrame)
      index为证券代码，类型为字符型(str)；columns为交易月份/日期，类型为字符型(str)。

  4. 例子

    • 输入

      import pandas as pd
import sys
sys.path.append('modules')
from factorProcess import *

factor = pd.read_csv('factor/IVCAPM.csv').set_index('secIDs')

afterWinsorize = winsorize(factor, type='Boxplot', times=1)
afterNeutralize = neutralize(afterWinsorize)
afterNormalize = normalize(afterNeutralize + 1)
print(afterNormalize)
