quoteclass.get_data()

data = quoteClass.get_data(secIDs=secIDs, datse=dates, fields=['Close', 'Volume'], showProcess=True)

  1. 释义

    获取目标证券池在一定日期范围内的日内行情数据

  2. 参数

    secIDs

    • 释义
      希望查询的证券代码
    • 类型
      序列(pd.Series) 或 列表(list)

    dates

    • 释义
      希望查询的日期范围
    • 类型
      字符型(str)

    fields

    • 释义
      希望查询的信息，目前支持查询的信息代码见5.代码
    • 类型
      序列(pd.Series) 或 列表(list)

    showProcess (optional)

    • 释义
      是否显示进度条
    • 类型
      布尔型(bool)

  3. 返回

    • 释义
      目标证券池在给定日期内的行情数据
    • 类型
      数据框(pd.DataFrame)
      按时间顺序从前到后排列。index为证券代码，类型为字符串(str)；每列为获取的数据，其中第一列默认为日期数据，第二列默认为日内分钟线排序(第n根k线)数据。

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getDate import dateclass
from getStock import stockclass
from getQuote import quoteclass

beginDate = '20050101'
endDate = '20050531'

dateClass = dateclass()
stockClass = stockclass('data/stock/1day', preRead=[])

dates = dateClass.get_dates(beginDate, endDate)
stockPools = stockClass.get_stockPools(dates)

quoteClass = quoteclass('data/stock/5min')

data = quoteClass.get_data(secIDs=stockPools[dates[0]], dates=dates, fields=['Close', 'Volume'])
print(data)
