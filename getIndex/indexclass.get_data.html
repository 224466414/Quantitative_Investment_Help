indexclass.get_data()

data = indexClass.get_data(field, date='All', idxID='All')

  1. 释义

    获取指定类型的指数数据，并可根据日期与证券进行筛选 需要注意的是，指数数据必须在创建indexclass对象时预读取

  2. 参数

    field

    • 释义
      希望查询的信息。具体代码详见5.代码
    • 类型
      字符型(str)

    date

    • 释义
      希望查询的日期，默认为All（所有日期）。具体代码详见5.代码
      必须传入日期序号（而不是日期数）
    • 类型
      整型(int), 列表(list), 一维数组(np.ndarray) 或 序列(pd.Series)

    idxID

    • 释义
      希望查询的指数，默认为All（所有日期）
      必须传入指数序号（而不是指数代码）
    • 类型
      整型(int), 列表(list), 一维数组(np.ndarray) 或 序列(pd.Series)

  3. 返回

    • 释义
      指定类型的指数数据
    • 类型
      二维数组(np.ndarray)，行按日期排列，列按指数排列

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getIndex import indexclass

# 创建数据获取对象
indexClass = indexclass(preReadFields=['Close'])

# 获取指定日期的数据
data = indexClass.get_data(field='Close', date=1000)
print('指定日期后获得的数据为:\n', data, '\n')

# 获取指定指数的数据
data = indexClass.get_data(field='Close', idxID=10)
print('指定指数后获得的数据为:\n', data, '\n')

# 获取指定日期与指数范围的数据
data = indexClass.get_data(field='Close', date=[i for i in range(0, 100)], idxID=[1, 2, 3, 4, 5])
print('同时指定日期与指数后获得的数据为:\n', data, '\n')

# 如果不传入任何日期与指数，则默认返回全部数据
data = indexClass.get_data(field='Close')
print('不指定任何日期与指数后获得的数据为:\n', data, '\n')
