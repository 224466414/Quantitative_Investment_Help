indexclass.get_infos()

infos = indexClass.get_infos(indexID)

  1. 释义

    获取目标指数的简要信息

  2. 参数

    indexID

    • 释义
      需要获取信息的指数代码
    • 类型
      整型(int), 列表(list), 一维数组(np.ndarray) 或 序列(pd.Series)

  3. 返回

    • 释义
      查询到的指数信息
    • 类型
      数据框(pd.DataFrame)
      其键为指数代码，类型为字符串(str)；其值为指数信息。

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getIndex import indexclass

# 创建数据获取对象
indexClass = indexclass()

# 传入指数序号得到的结果
names = indexClass.get_infos(0)
print('传入指数序号得到的结果:\n', names, '\n')

# 传入指数序号列表得到的结果
names = indexClass.get_infos([0, 1])
print('传入指数序号列表得到的结果:\n', names, '\n')

# 传入指数代码得到的结果
names = indexClass.get_infos('000001.SH')
print('传入指数代码得到的结果:\n', names, '\n')

# 传入指数代码列表得到的结果
names = indexClass.get_infos(['000001.SH', '000002.SH'])
print('传入指数代码列表得到的结果:\n', names, '\n')
