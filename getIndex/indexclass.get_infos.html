indexclass.get_infos()

infos = indexClass.get_infos(indexIDs, showProcess=True)

  1. 释义

    获取目标证券的证券简称

  2. 参数

    indexIDs

    • 释义
      需要获取信息的指数代码
    • 类型
      序列(pd.Series) 或 列表(list)

    showProcess (optional)

    • 释义
      是否显示进度条
    • 类型
      布尔型(bool)

  3. 返回

    • 释义
      查询到的指数信息
    • 类型
      数据框(pd.DataFrame)
      其键为指数代码，类型为字符串(str)；其值为指数信息。

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getIndex import indexclass

indexClass = indexclass('data/index', preRead=['Close', 'Volume'])

indexIDs = ['000001.SH']

infos = indexClass.get_infos(indexIDs)
print(infos)

    • 输出
