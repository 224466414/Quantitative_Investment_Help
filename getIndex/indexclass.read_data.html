indexclass.read_data()

fileContent = indexClass.read_data(filename, index=True)

  1. 释义

    读取指定数据文件。例如，读取 "indexClose.npy" 中的数据，该数据文件包含所有指数在所有日期的收盘价。

  2. 参数

    filename

    • 释义
      要求读取的文件名称。不带后缀，不需要指定文件夹
    • 类型
      字符型(str)

    index (optional)

    • 释义
      读取后是否加载表头。默认为True。index为交易日期，column为股票代码
    • 类型
      布尔型(bool)

  3. 返回

    • 释义
      返回读取的数据
    • 类型
      数据框(pd.DataFrame)

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getIndex import indexclass

indexClass = indexclass('data/index', preRead=['Close', 'Volume'])

fileName = 'indexClose'
fileContent = indexClass.read_data(fileName)
print(fileContent)
