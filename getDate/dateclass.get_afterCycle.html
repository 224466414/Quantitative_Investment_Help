dateclass.get_afterCycle()

afterCycle = dateClass.get_afterCycle(currentCycle, cycleSpan, showProcess=True)

  1. 释义

    获取指定周期后n周期的周期数。例如，在周期长度为20的情况下，导出第50号周期之后的第20个周期。

  2. 参数

    currentCycle

    • 释义
      指定的当前周期数
    • 类型
      整型(int)

    cycleSpan

    • 释义
      向后计算周期数的时间跨度
    • 类型
      整型(int)

    showProcess (optional)

    • 释义
      是否显示进度条
    • 类型
      布尔型(bool)

  3. 返回

    • 释义
      指定周期后n周期的周期数
    • 类型
      整型(int)

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getDate import dateclass

dateClass = dateclass(cycleLength=20)

currentCycle = 50
cycleSpan = 20

afterCycle = dateClass.get_afterCycle(currentCycle, cycleSpan)
print(afterCycle)

    • 输出

      70
