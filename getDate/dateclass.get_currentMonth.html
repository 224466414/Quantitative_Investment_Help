dateclass.get_currentMonth()

currentCycle = dateClass.get_currentMonth(currentDate)

  1. 释义

    获取指定日期对应的月份序号。例如，获取2006年12月29日的月份序号。

  2. 参数

    currentDate

    • 释义
      指定的当前日期
      可传入日期数（如20050104）或日期序号（如199对应第200天）
    • 类型
      整型(int)

  3. 返回

    • 释义
      指定日期对应的月份序号
    • 类型
      整型(int)

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getDate import dateclass

dateClass = dateclass()

# 传入日期数参数
currentDate = 20061220
currentMonth = dateClass.get_currentMonth(currentDate)
print('传入日期数参数的结果为:\n', currentMonth, '\n')

# 传入日期序号数参数
currentMonth = 900
currentMonth = dateClass.get_currentMonth(currentDate)
print('传入日期序号参数的结果为:\n', currentMonth, '\n')

    • 输出

      传入日期数参数的结果为:
23

传入日期序号参数的结果为:
23
