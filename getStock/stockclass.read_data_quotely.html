stockclass.read_data_quotely()

fileContent = stockClass.read_data_quotely(fileName, type='5min')

  1. 释义

    读取指定数据文件。例如，读取 "StockClose5M_20050104.mat" 中的数据，该数据文件包含所有证券在该日期的5分钟k线收盘价。

  2. 参数

    fileName

    • 释义
      要求读取的文件名称，需要带后缀
    • 类型
      字符型(str)

    type (optional)

    • 释义
      读取的数据类型，默认为5min（5分钟K线数据）
      当前支持5min（5分钟K线数据）与60min（60分钟K线数据）
    • 类型
      字符型(str)

  3. 返回

    • 释义
      要求读取的文件内容
    • 类型
      二维数组(np.ndarray)，行按K线顺序排列，列按证券排列

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getStock import stockclass

stockClass = stockclass()

fileName = 'StockClose5M_20050104.mat'
fileContent = stockClass.read_data_quotely(fileName, type='5min')
print(fileContent)
