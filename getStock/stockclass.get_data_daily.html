stockclass.get_data_daily()

data = stockclass.get_data_daily(field, date='All', secID='All')

  1. 释义

    获取指定类型的日线数据，并可根据日期与证券进行筛选
    需要注意的是，日线数据必须在创建stockclass对象时预读取

  2. 参数

    field

    • 释义
      希望查询的信息。具体代码详见5.代码
    • 类型
      字符型(str)

    date

    • 释义
      希望查询的日期，默认为All（所有日期）
      必须传入日期序号（而不是日期数）
    • 类型
      整型(int), 列表(list), 一维数组(np.ndarray) 或 序列(pd.Series)

    secID

    • 释义
      希望查询的证券，默认为All（所有日期）
      必须传入证券序号（而不是证券代码）
    • 类型
      整型(int), 列表(list), 一维数组(np.ndarray) 或 序列(pd.Series)

  3. 返回

    • 释义
      指定类型的日线数据
    • 类型
      二维数组(np.ndarray)，行按日期排列，列按证券排列

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getDate import dateclass
from getStock import stockclass

# 创建数据获取对象
stockClass = stockclass(preReadDaily=['CloseR'])

# 获取指定日期的数据
data = stockClass.get_data_daily(field='CloseR', date=100)
print('指定日期后获得的数据为:\n', data, '\n')

# 获取指定证券的数据
data = stockClass.get_data_daily(field='CloseR', secID=100)
print('指定证券后获得的数据为:\n', data, '\n')

# 获取指定日期与证券范围的数据
dateClass = dateclass()
dates = dateClass.get_dates(20050101, 20210531)
stockPools = stockClass.get_stockPools(dates)
secIDs = stockPools[dates[0]]
data = stockClass.get_data_daily(field='CloseR', date=dates, secID=secIDs)
print('同时指定日期与证券后获得的数据为:\n', data, '\n')

# 如果不传入任何日期与证券，则默认返回全部数据
data = stockClass.get_data_daily(field='CloseR')
print('不指定任何日期与证券后获得的数据为:\n', data, '\n')
