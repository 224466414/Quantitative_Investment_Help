量化投资程序 帮助文档

stockclass.get_data()

data = stockclass.get_data(secIDs=stockPool, dates=dates, attributes=['CloseR', 'Volume'], showProcess=True)

  1. 释义

    获取目标证券池在一定日期范围内的行情数据

  2. 参数

    secIDs

    • 释义
      希望查询的证券代码
    • 类型
      序列(pd.Series) 或 列表(list)

    dates

    • 释义
      希望查询的日期序列
    • 类型
      序列(pd.Series) 或 列表(list)

    fields

    • 释义
      希望查询的信息。具体代码详见5.代码
    • 类型
      序列(pd.Series) 或 列表(list)

    showProcess (optional)

    • 释义
      是否显示进度条
    • 类型
      布尔型(bool)

  3. 返回

    • 释义
      目标证券池在一定日期范围内的行情数据
    • 类型
      数据框(pd.DataFrame)
      按时间顺序从前到后排列。index为证券代码，类型为字符串(str)；每列为获取的数据，其中第一列默认为日期数据。

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getDate import dateclass
from getStock import stockclass

dateClass = dateclass('data/date')
stockClass = stockclass('data/stock/1day', preRead=['CloseR', 'Volume'])

beginDate = '20050101'
endDate = '20210531'
dates = dateClass.get_dates(beginDate, endDate)

stockPools = stockClass.get_stockPools([dates[0]])
stockPool = stockPools[dates[0]]

data = stockClass.get_data(secIDs=stockPool, dates=dates.iloc[0:100], fields=['CloseR', 'Volume'])
print(data)
