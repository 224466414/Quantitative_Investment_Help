stockclass.get_stockPools()

stockPools = stockclass.get_stockPools(date, delST=True, delStop=True, delNew=True, minDate=60)

  1. 释义

    获取所有指定日期的，经过去ST、去停牌、去次新股以后的证券池

  2. 参数

    date

    • 释义
      获取证券池的日期
    • 类型
      整型(int), 列表(list), 一维数组(np.ndarray) 或 序列(pd.Series)

    delST (optional)

    • 释义
      是否需要剔除ST股票。默认为True
    • 类型
      布尔型(bool)

    delStop (optional)

    • 释义
      是否需要剔除停牌股票。默认为True
    • 类型
      布尔型(bool)

    delNew (optional)

    • 释义
      是否需要剔除次新股。默认为True
    • 类型
      布尔型(bool)

    minDate (optional)

    • 释义
      次新股的定义，上市多少个交易日内定义为次新股。默认为60
    • 类型
      整型(int)

  3. 返回

    • 释义
      给定月份的所有交易日
    • 类型
      若输入的日期为整型(int)，则返回一维数组(np.ndarray) 若输入的日期为列表(list)、一维数组(np.ndarray)、序列(pd.Series)，则返回字典(dict)。其键为日期序号，类型为整型(int)；其值为证券池，类型为一维数组(np.ndarray)

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getStock import stockclass

# 创建数据获取对象
stockClass = stockclass()

# 仅输入一个日期时
stockPool = stockClass.get_stockPools(100)
print('仅输入一个日期时获得的数据为:\n', stockPool, '\n')

# 输入多个日期时
stockPool = stockClass.get_stockPools([100, 101, 102])
print('输入多个日期时获得的数据为:\n', stockPool, '\n')

    • 输出

      仅输入一个日期时获得的数据为:
[   1    2    3 ... 1376 1377 1378] 

输入多个日期时获得的数据为:
{100: array([   1,    2,    3, ..., 1376, 1377, 1378], dtype=int64),
101: array([   1,    2,    3, ..., 1376, 1377, 1378], dtype=int64),
102: array([   1,    2,    3, ..., 1376, 1377, 1378], dtype=int64)}
