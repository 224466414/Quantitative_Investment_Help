stockclass.read_data()

fileContent = stockclass.read_data(filename, index=True, factor=False)

  1. 释义

    读取指定数据文件。例如，读取 "stockClose.npy" 中的数据，该数据文件包含所有证券在所有日期的收盘价。
    使用读取跨日行情数据时要考虑是否复权。该函数使用前复权因子对行情数据进行复权。

  2. 参数

    filename

    • 释义
      要求读取的文件名称。不带后缀，不需要指定文件夹
    • 类型
      字符型(str)

    index (optional)

    • 释义
      读取后是否加载表头。默认为True。index为交易日期，column为股票代码
    • 类型
      布尔型(bool)

    factor (optional)

    • 释义
      读取后是否进行复权。默认为False。该函数使用前复权因子对行情数据进行复权
    • 类型
      布尔型(bool)

  3. 返回

    • 释义
      返回读取的数据
    • 类型
      数据框(pd.DataFrame)

  4. 例子

    • 输入

      import sys
sys.path.append('modules')
from getStock import stockclass

stockClass = stockclass('data/stock/1day', preRead=['CloseR', 'Volume'])

fileName = 'stockClose'
fileContent = stockClass.read_data(fileName)
print(fileContent)
